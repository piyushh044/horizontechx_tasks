# 📈 Stock Trading Platform

A simple terminal-based stock trading simulation written in **Java**.  
Buy and sell stocks, view live market prices.

---

## ✨ Features

- **Predefined stock market** – AAPL, GOOG, AMZN, TSLA with realistic starting prices
- **Dynamic price updates** – prices change randomly every time.
- **Buy stocks** – purchase shares if you have enough cash
- **Sell stocks** – sell shares from your portfolio.
- **Portfolio summary** – shows cash, each holding, and total value
- **Simple user interface** – numbered menus for all actions
